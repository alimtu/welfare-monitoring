# University Welfare Monitoring

A Next.js (App Router) PWA for university welfare monitoring — dynamic form building,
submission management, and related services.

## Getting started

```bash
pnpm install
pnpm dev      # runs on http://localhost:3001
```

## Scripts

- `pnpm dev` — start the dev server (Turbopack, port 3001)
- `pnpm build` — production build
- `pnpm start` — run the production build
- `pnpm lint` — run Next.js lint

## Tech stack

- Next.js 16 (App Router) + React 19
- Tailwind CSS 4 + shadcn/ui
- TanStack Query, React Hook Form
- next-pwa (offline support)
